# Pakistan Weather Forecast

## 🌤️ Real-Time Weather Dashboard

**Pakistan Cities** | **Live APIs** | **Dynamic Icons** | Glassmorphism UI | WeatherStack + OpenWeather powered forecast.

**Live Demo**: [Open index.html](index.html)

### ✨ Features

- **🏙️ Pakistan Focus** - 5 quick cities + search
- **🌡️ Live Data** - Temperature, feels-like, humidity, wind, pressure
- **🎨 Dynamic Icons** - Sun/Cloud/Rain/Thunder/Snow auto-matching
- **📱 Quick Access** - One-click major cities (Karachi, Lahore, Islamabad...)
- **💎 Glassmorphism** - Backdrop blur + gradients
- **📐 Responsive Grid** - Mobile/desktop optimized
- **🚀 Real-time** - Updates on search (no refresh)

### 🚀 Quick Start

1. **Open index.html** → **Search city** or **Quick button**
2. **Live weather** loads instantly
3. **Icons update** by description

```
Karachi → 32°C ☀️ Feels 35°C | Humidity 60%
Lahore → 28°C 🌧️ Wind 15kmh | Pressure 1012hPa
```

### 🛠️ Live APIs

```
Geocoding: OpenWeather (PK: 37f1f49148144cfed9af6e2cd96026bc)
Weather: WeatherStack (PK: 3e1149f6595613215cbfbc55859df93d)

Flow: City → lat/lon → weather data → UI update
```
